# مشروع EDOCS - نظام تطوير وتحسين المنتجات الفوارة (Effervescent Optimization System)

![EDOCS](https://img.shields.io/badge/Status-Active-success?style=for-the-badge) ![Language](https://img.shields.io/badge/Language-Arabic/English-green?style=for-the-badge)

## 📋 نظرة عامة
نظام **EDOCS** هو منصة صيدلانية متطورة مصممة خصيصاً لإدارة عمليات تطوير المنتجات الفوارة. يهدف النظام إلى رقمنة وتوحيد سير العمل من مرحلة النماذج الأولية إلى الإنتاج التجاري، مع تركيز خاص على دقة التركيبات وإدارة المخاطر.

---

## ✨ المميزات الرئيسية

### 1. 🏗️ إدارة دورة حياة المشروع
*   تتبع كافة مراحل تطوير المنتج الصيدلاني.
*   إدارة البيانات الوصفية للمشروع والأسواق المستهدفة.
*   واجهات مستخدم مرنة تدعم اللغتين العربية والإنجليزية.

### 2. 🧪 محرك التركيبات والصيغ
*   إدارة فئات المواد الفعالة (APIs) والمواد المضافة (Excipients).
*   خرائط تفصيلية للمكونات داخل كل صيغة صيدلانية.
*   حساب تكاليف الوحدات ونسب المكونات.

### 3. 📊 العمليات التجريبية
*   تسجيل كافة التجارب المخبرية مع توثيق المعدات المستخدمة.
*   متابعة حجم التشغيل ومسارات الإنتاج.
*   قاعدة بيانات مركزية للنتائج التاريخية للرجوع إليها مستقبلاً.

### 4. 🛡️ نظام إدارة المخاطر (FMEA)
*   تحديد "أوضاع الفشل" (Failure Modes) المحتملة.
*   **حساب تلقائي لـ RPN:** يقوم النظام بضرب الشدة × الحدوث × الاكتشاف ليعطيك درجة الخطورة.
*   **التنبيه اللوني:** تمييز المخاطر العالية باللون الأحمر، المتوسطة بالأصفر، والمنخفضة بالأخضر.
*   توثيق إجراءات التخفيف وتاريخ المراجعة والمسؤول عن كل خطر.

## 🛠️ التقنيات المستخدمة
*   **الإطار البرمجي:** Laravel 11/12.
*   **لوحة التحكم:** Filament PHP (Dashboard).
*   **التفاعليات:** Livewire (للحسابات التلقائية في الوقت الفعلي).
*   **قاعدة البيانات:** MySQL.

## ⚙️ طريقة التشغيل والإعداد

1.  تأكد من وجودك في مجلد المشروع الرئيسي: `c:\Users\Dell\Desktop\EDOS\edocs`
2.  قم بتشغيل الخادم المحلي:
    ```powershell
    php artisan serve
    ```
3.  قم بتحديث قاعدة البيانات (في حال إضافة حقول جديدة):
    ```powershell
    php artisan migrate
    ```
4.  روابط الدخول:
    *   **الصفحة الرئيسية:** [http://127.0.0.1:8000](http://127.0.0.1:8000)
    *   **لوحة التحكم:** [http://127.0.0.1:8000/admin](http://127.0.0.1:8000/admin)
